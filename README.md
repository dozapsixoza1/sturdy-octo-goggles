# 🎰 GRAM Рулетка — Telegram Bot

## Установка

1. Установи зависимости:
```
pip install -r requirements.txt
```

2. Открой `config.py` и замени:
   - `BOT_TOKEN` — токен от @BotFather
   - `ADMIN_IDS` — твой Telegram user_id (узнай у @userinfobot)

3. Запусти:
```
python bot.py
```

## 🎮 Команды игрока

| Команда | Описание |
|---|---|
| `100 к` | Ставка на красное x2 |
| `100 ч` | Ставка на чёрное x2 |
| `100 з` | Ставка на зеро x36 |
| `100 17` | Ставка на число x36 |
| `100 чт` | Ставка на чёт x2 |
| `100 нч` | Ставка на нечет x2 |
| `100 д1` | Дюжина 1-12 x3 |
| `100 д2` | Дюжина 13-24 x3 |
| `100 д3` | Дюжина 25-36 x3 |
| `100 м` | Меньше 1-18 x2 |
| `100 б` | Больше 19-36 x2 |
| `/balance` | Баланс и статистика |
| `/history` | История ставок |
| `/promo КОД` | Активировать промокод |

## 👑 Панель администратора

| Команда | Описание |
|---|---|
| `/admin` | Открыть панель |
| `/users` | Список всех игроков |
| `/userinfo [id]` | Инфо об игроке |
| `/addbalance [id] [сумма]` | Добавить баланс |
| `/setbalance [id] [сумма]` | Установить баланс |
| `/block [id]` | Заблокировать |
| `/unblock [id]` | Разблокировать |
| `/stats` | Статистика казино |
| `/allbets` | Последние ставки |
| `/setminbet [сумма]` | Мин. ставка |
| `/setmaxbet [сумма]` | Макс. ставка |
| `/createpromo [код] [сумма] [использований]` | Создать промокод |
| `/promos` | Список промокодов |
| `/delpromo [код]` | Деактивировать промокод |

## 📁 Структура файлов

```
roulette_bot/
├── bot.py          — запуск бота
├── config.py       — токен и настройки
├── database.py     — база данных SQLite
├── roulette.py     — логика игры
├── requirements.txt
└── handlers/
    ├── user.py     — команды игроков
    └── admin.py    — панель администратора
```
